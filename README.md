# 🛡️ Informe de Laboratorio – Ethical Hacking en Entorno Virtual

## 📌 Descripción
Este repositorio documenta el desarrollo de una práctica de **Ethical Hacking** realizada en un entorno virtual controlado. Se evidencia la comprensión de conceptos de ciberseguridad, el uso de herramientas de análisis de vulnerabilidades y la interpretación técnica de los resultados obtenidos.

---

## 📖 7.2 Introducción

El **Ethical Hacking** es una disciplina clave dentro de la ciberseguridad que busca identificar vulnerabilidades en sistemas informáticos mediante técnicas controladas y autorizadas, con el objetivo de prevenir ataques reales.

Las **pruebas de penetración (Pentesting)** permiten simular ataques para descubrir fallos de seguridad en sistemas, redes y aplicaciones.

### 🔍 Fases del Pentesting
- **Reconocimiento:** Recolección de información del objetivo.
- **Escaneo:** Identificación de puertos y servicios.
- **Explotación:** Uso de vulnerabilidades para obtener acceso.
- **Post-explotación:** Análisis del sistema comprometido.

Las pruebas deben realizarse en **entornos controlados** para evitar impactos en sistemas reales.

### 🛠️ Herramientas utilizadas
- Nmap (escaneo de red)
- Metasploit (explotación de vulnerabilidades)
